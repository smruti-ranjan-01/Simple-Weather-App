#  Simple Weather App

A lightweight and beginner-friendly weather application that displays real-time weather information for any city using a weather API.

##  Features
- Search weather by city name
- Displays temperature, weather condition, humidity, and wind speed
- Clean and responsive UI
- Fast API data fetching

##  Tech Stack
-React
- Weather API (e.g., OpenWeatherMap)

##  How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Enter a city name to get weather details

##  Purpose
This project is built for learning API integration, asynchronous JavaScript, and basic frontend development.
